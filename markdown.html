Markdown

Markdown är ett märkspråk för ren text som sedan kan konverteras till många olika format (pdf, HTML, wordfil). Syftet med markdown är att man enkelt ska kunna läsa och skriva läsbar ren text, men med möjligheten att sedan kunna konvertera texten till strukturerad, rik text. Syntaxen för att formatera texten för fet och kursiv stil, rubriker och punktlistor är förenklad jämfört med html-kod. Där man i html skriver

Rubriktext
